# Color Generator
A Scrimba project challenge.

The goal of this project was to utilize [The Color API](https://www.thecolorapi.com/)

## Features
- Choose a color with the color picker, or refresh the page to get a random color.
- Choose from 5 different color scheme methods.
- Copy the hex codes of generated colors.

## Methods
- REST API call to to The Color API endpoint using dynamic parameters.
- For loop for random hex code generation on page load before executing the API call.
